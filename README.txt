== Description ==
Integra en métodos de envio el servicio de WSDL de Coordinadora Colombia, para calcular los costos de envío automáticamente.

== Installation ==
1. Upload "coordiplug.zip" to the "/wp-content/plugins/" directory and extract.
2. Activate the plugin through the "Plugins" menu in WordPress.
